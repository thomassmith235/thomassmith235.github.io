FMECA Web App Support

Welcome! This page covers the user manual, FAQs, and support contact details for the FMECA Web App.

If you need help, email us at contact@nevis.tools.

User Manual

1) Create an account and sign in

  • Open the app and choose Create account.
  • Confirm your email using the link from Supabase.
  • Sign in to access your projects.

2) Create a new project

  • Click + New project in the left sidebar.
  • Enter a Project title and Description.

3) Build your FMECA

  • Add Subsystems.
  • Inside each subsystem, add Components.
  • Inside each component, add Failure Modes with:
    • Title
    • Effect
    • Severity (1-10)
    • Occurrence (1-10)
    • Detection (1-10)

RPN is calculated automatically: Severity x Occurrence x Detection.

4) Save your project

  • Click Save project in the header.
  • The status line shows the last saved time.

5) Export

  • Click Export and choose:
    • Export to Excel (CSV)
    • Export to PDF (opens print dialog in a new tab)

MVP Limits (Free Tier)

  • 1 project per user
  • Restricted number of subsystems, components, and failure modes per project
  • Up to 10 components per subsystem
  • Up to 10 failure modes per component

When a limit is exceeded the app will show a message indicating the free tier limit. Paid tier is coming soon.

FAQs

Q: Why can't I save additional projects or add more items?
A: The MVP free tier has restricted project and item limits. Paid tier is coming soon.

Q: My Save button fails. What should I check?
A: Confirm you're signed in and your API is reachable. If limits are exceeded, the app will show a clear message.

Q: I see "Session expired. Please sign in again."
A: Your token expired. Sign in again to continue.

Q: Why does PDF export open a new tab?
A: It uses the browser's print dialog. Allow pop-ups for the site to export PDFs.

Q: Where is my data stored?
A: Projects are stored in the app database associated with your account. See the Privacy section below.
